Trabalho desenvolvido na discplina de PROGRAMAÇÃO WEB desenvolvido pelos alunos: Gustavo de Moura, Henrique e João Zanona;
O objetivo da atividade era desenvolver um site para uma construtora, onde fossem disponibilizados imóveis para a venda, além da prestação de serviços para construções, incluindo também o contato direto com a empresa;
Foi buscado desenvolver um site no estilo profissional e realista, com simplicidade, mas também funcionalidade e praticidade.
